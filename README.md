# SafeNPM Client

## Get started developing
```
git clone https://github.com/ISNIT0/safe-npm-client.git
cd safe-npm-client
npm i
npm run start
open http://localhost:8080
```

## Developing
```
npm i
npm run start
open http://localhost:8080

open http://localhost:8080/about
open http://localhost:8080/package/axios
```
